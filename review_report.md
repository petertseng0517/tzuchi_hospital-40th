# 花蓮慈濟醫院40周年院慶專網 — 高層審視報告

> 角色定位：院方最高籌備主責，兼顧對外形象、院內凝聚、技術可行性與活動效益
>
> 審視日期：2026-03-27

---

## 一、現況總評

本專案已完成一個**技術扎實、主視覺統一**的靜態紀念網站，具備：

| 項目 | 評估 |
|------|------|
| 視覺設計 | 信任藍 + 慶典金，符合慈濟形象 |
| 技術架構 | 純靜態，無伺服器依賴，維護成本極低 |
| 效能優化 | WebP、Lazy Load、音訊 preload=none，表現良好 |
| 無障礙設計 | ARIA、鍵盤導覽、動態偏好設定均已落實 |
| 互動深度 | 時間軸、相片燈箱、翻頁書、AI語音導覽 |

**整體評語**：這是一個「夠用但未滿」的版本。技術底子好，但在**情感感召力**、**院內外連結**與**活動整合**上仍有明顯空間。

---

## 二、改善建議（依優先序）

### 優先級一：強化情感連結（上線前必須完成）

**1. 英雄區 (Hero) 缺少動態感**
- 現況：靜態背景圖 + 固定文字
- 建議：加入**數字跳動計數器**（已服務 X 萬名病患、X 位醫護、X 年守護），在頁面載入時從0計數至目標值
- 預期效果：立即建立規模感，訪客5秒內感受40年厚度

**2. 時間軸節點過少（僅3個）**
- 現況：1986 / 2006 / 2026，跨度大、空洞感強
- 建議：補充**8-10個關鍵里程碑**（首位住院病患、骨髓移植成功首例、921震災醫療支援、COVID-19防疫貢獻、JCI認證等）
- 預期效果：訪客停留時間預估增加 40-60 秒

**3. 缺乏真人故事**
- 現況：文字與圖片為主，無人物敘事
- 建議：新增「**守護者的話**」小單元，放置 3-5 位資深醫護/創院元老的**短片或語音片段**（30-60秒），搭配黑白到彩色的照片轉場
- 預期效果：最高情感共鳴點，社群分享率預估提升 2-3 倍

---

### 優先級二：擴大觸及與傳播（活動前2週上線）

**4. 缺乏社群分享機制**
- 現況：無任何分享按鈕
- 建議：新增 **Open Graph meta tags**（填入標題、描述、og:image），以及 LINE / Facebook 分享按鈕
- 花蓮地區 LINE 使用率極高，此為最重要傳播渠道
- 預期效果：自然流量透過員工分享擴散，觸及院外家屬與社區

**5. 缺少40周年活動資訊頁**
- 現況：網站僅是紀念性，無活動導引功能
- 建議：新增「**慶典活動**」區塊，列出院慶當日的活動時程、地點地圖、報名/預約連結
- 預期效果：網站從「被動閱覽」轉為「主動導流」工具

**6. 歷史相片區資訊不足**
- 現況：41張老照片，caption 僅顯示年份
- 建議：為每張照片補充**一句話說明**（地點、事件、人物），由院史委員會提供內容，工讀生輸入
- 預期效果：老員工認出自己的歷史記憶，大幅提升院內黏著度

---

### 優先級三：長期維運與數位資產（活動後持續）

**7. 翻頁書在手機上隱藏**
- 現況：`@media (max-width: 768px)` 下翻頁書區塊被隱藏（DearFlip 行動裝置體驗不佳）
- 問題：超過 60% 流量來自手機，訪客遇到空白區塊
- 建議：手機版改為**PDF 直接下載按鈕 + 封面預覽圖**，維持功能完整性
- 預期效果：手機訪客不再遇到空白區塊，完整使用率從 0% 提升至 100%

**8. jQuery 版本過舊（1.11.0，2014年版）**
- 現況：DearFlip 依賴 jQuery 1.11.x，有已知安全漏洞
- 建議：確認 DearFlip 是否支援 jQuery 3.x，若支援請升級；若不支援，限制 jQuery 作用域於 DearFlip 範圍內
- 預期效果：降低 XSS 風險，符合資安合規要求

**9. 缺乏流量追蹤**
- 現況：無任何分析工具
- 建議：嵌入 **Google Analytics 4** 或院內自建的 Matomo（若有隱私考量），追蹤哪些內容最受矚目
- 預期效果：為下次院慶（45周年）的企劃提供數據依據

## 四、行動建議

### 立即可執行（1週內，技術團隊自行處理）

1. 補充 Open Graph meta tags（約1小時工作量）
2. 加入 LINE / Facebook 分享按鈕（約2小時）
3. 手機版翻頁書改為 PDF 下載按鈕（約半天）
4. 升級或隔離 jQuery 1.11.0（約半天）

### 需要院方配合提供素材

| 負責單位 | 所需素材 |
|---------|---------|
| 院史委員會 | 時間軸補充里程碑內容（8-10條） |
| 院史委員會 | 41張歷史相片的說明文字 |
| 公關室 | 醫護真人故事文字或影音（3-5則） |
| 院慶籌備組 | 活動時程表、地點、報名連結 |

### 建議最終上線時間

院慶活動前 **2 週**，搭配下列管道同步推播：
- 全院 Email 通知
- 院方 Facebook 官方帳號
- 院方 LINE 官方帳號

## 五、結語

本專網的技術基礎非常扎實，真正缺的是**院方內容的投入**。程式面的改善可隨時動手推進，最關鍵的瓶頸在於**院史素材的蒐集與授權確認**。建議院方盡早啟動素材蒐集流程，確保網站能在院慶前以完整面貌呈現，達到凝聚院內向心力、對外展現40年品牌價值的雙重目標。
